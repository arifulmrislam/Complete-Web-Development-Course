Important Question's For Job Interview!!!

1. What is DOM, and what is the purpose of DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

2.How will you select an HTML Element using DOM? Name the DOM Methods.

DOM Selectors, as the name suggests, are used to select HTML elements within a document using JavaScript. There are 5 ways in which you can select elements in a DOM using selectors.

  1. getElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

3. What is Event Bubble?

Event bubbling is a method of event propagation in the HTML DOM API when an event is in an element inside another element and both elements have registered a handle to that event. It is a process that starts with the element that triggered the event and then bubbles up to the containing elements in the hierarchy.

4. What is A callback function and why will you use it?

A JavaScript callback is a function that is to be executed after another function has finished execution. A more formal definition would be - Any function that is passed as an argument to another function so that it can be executed in that other function is called a callback function.
